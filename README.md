# Selenium C# Automation Framework

This project is a Test Automation Framework built using:

- Selenium WebDriver
- C#
- MSTest
- Page Object Model (POM)
- Extent Reports
- XML Data-Driven Testing

## Project Features

- Login Test Cases (Valid & Invalid)
- Product Selection
- Cart Functionality
- Checkout Flow
- Negative Validations
- Screenshot Capture
- Extent Reporting

## Framework Design

- BasePage
- Page Object Model Structure
- Data-Driven Testing (XML)
- Exception Handling
- Screenshot on Pass/Fail
